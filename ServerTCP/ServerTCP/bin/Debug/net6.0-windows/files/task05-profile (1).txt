# Інформація про поточного користувача

Додати на сторінках в навігації вивід інформації про поточного користувача
(логін)
Додати кнопку вихід на кнопку профіль
Профіль користувача виводити в модальному вікні

Для не авторизованого користувача виводити у навігації кнопкит/посилання для входу або реєстрації
